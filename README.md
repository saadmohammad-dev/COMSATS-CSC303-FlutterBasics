# CSC303: Mobile Application Development — Week 6 Labs

This repository contains the Flutter implementations for the Week 6 Lab tasks at COMSATS University Islamabad, Abbottabad Campus.

## 🚀 Tasks Covered:
* **Q1:** Asset Image Loading (`Image.asset` configuration)
* **Q2:** Grade Calculator using Widget Constructors
* **Q3:** Data passing between Widgets (Name & Attendance simulation)
* **Q4:** Interactivity basics (Buttons & Snackbars)
* **Q5 & Q6:** `TextField` manipulation, uppercase stream, and value assignment
* **Q7:** State toggling & disabling UI elements
* **Q8:** Registration Form validation using `TextEditingController`
* **Q9:** Basic Math Operation (Addition of inputs)
* **Q10:** String expression parsing (Mini Calculator Layout & Logic)

## 🛠️ Tech Stack
* Flutter (SDK)
* Dart
